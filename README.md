# Домашнее задание: Коллекции в C#

Решение включает реализацию трех задач по работе с коллекциями в C#.

## Структура проекта

- **Program.cs** - Основной файл с реализацией всех трех заданий
- **homework.csproj** - Файл конфигурации .NET проекта

## Описание заданий

### Задание 1: Работа со списком строк (List)

Реализована программа для работы со списком строк с возможностью:
- Добавления элементов в конец списка
- Вставки элементов в середину списка
- Вывода содержимого списка
- Цикличного добавления элементов до команды `exit`

**Класс:** `ListTask`

### Задание 2: Словарь студентов и их оценок (Dictionary)

Реализована программа для управления словарем студентов с возможностью:
- Добавления студента с проверкой корректности оценки (от 2 до 5)
- Поиска студента по имени и вывода его оценки
- Отображения всех студентов в словаре
- Выхода по команде `exit`

**Класс:** `DictionaryTask`

### Задание 3: Двусвязный список (DoublyLinkedList)

Реализована собственная реализация двусвязного списка с возможностью:
- Создания списка и добавления от 3 до 6 элементов
- Вывода списка в прямом порядке
- Вывода списка в обратном порядке
- Навигации по списку в обоих направлениях благодаря полям `Next` и `Prev`

**Класс:** `DoublyLinkedListTask` с вложенным классом `Node`
